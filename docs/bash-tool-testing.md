# Bash Tool Phase 1 测试指南

## 测试环境要求

1. 服务端运行在 localhost:3001
2. 客户端 Electron 应用已启动
3. 已连接到一个会话

## 手动测试步骤

### 测试 1: 基本命令执行

**步骤：**
1. 打开应用，进入聊天界面
2. 发送消息：`执行命令 echo "Hello World"`

**期望结果：**
- 助手调用 bash 工具
- 输出：`Hello World`
- 退出码：0

### 测试 2: cd 持久化（核心功能）

**步骤：**
1. 发送：`执行 pwd`（记录当前目录）
2. 发送：`执行 cd /tmp`
3. 发送：`执行 pwd`

**期望结果：**
- 第1步：显示初始目录（如 `/Users/xxx`）
- 第3步：显示 `/tmp`
- 目录变更已保持

### 测试 3: 环境变量保持

**步骤：**
1. 发送：`执行 export MY_TEST_VAR=hello123`
2. 发送：`执行 echo $MY_TEST_VAR`

**期望结果：**
- 第2步输出：`hello123`
- 环境变量在会话中保持

### 测试 4: 安全拦截

**步骤：**
1. 发送：`执行 rm -rf /`

**期望结果：**
- 命令被拦截
- 返回错误信息：`Dangerous command blocked: rm -rf /`
- 退出码：-1

### 测试 5: 超时控制

**步骤：**
1. 发送：`执行 sleep 10`（超时设为 5 秒）

**期望结果：**
- 约 5 秒后返回超时错误
- 错误信息包含：`[Timeout]`

### 测试 6: 输出截断

**步骤：**
1. 发送：`执行 seq 1 100000`（生成大量输出）

**期望结果：**
- 输出被截断（约 20 万字符限制）
- 最后显示：`... (truncated)`

### 测试 7: 多命令组合

**步骤：**
1. 发送：`执行 cd /tmp && mkdir test_dir_$$ && cd test_dir_$$ && pwd`

**期望结果：**
- 成功创建目录（$$ 是当前 PID）
- 显示完整路径：`/tmp/test_dir_XXX`

### 测试 8: 会话隔离

**步骤：**
1. 在会话 A 执行：`export SESSION_A_VAR=iam_a`
2. 切换到会话 B
3. 在会话 B 执行：`echo $SESSION_A_VAR`

**期望结果：**
- 会话 B 输出为空（变量未定义）
- 会话间环境隔离

### 测试 9: 应用退出清理

**步骤：**
1. 执行几个命令，保持会话活跃
2. 关闭应用（Cmd+Q）
3. 检查系统进程：`ps aux | grep "bash -i"`

**期望结果：**
- 没有残留的 `bash -i` 进程
- 应用退出时自动清理

## 测试检查清单

- [ ] 基本命令执行正常
- [ ] cd 命令持久化有效
- [ ] 环境变量跨调用保持
- [ ] 危险命令被拦截
- [ ] 超时机制工作正常
- [ ] 大输出被截断
- [ ] 多命令组合执行正确
- [ ] 会话间环境隔离
- [ ] 应用退出清理完成

## 已知限制

1. **无 PTY 支持**：交互式程序（vim, top）可能无法正常工作
2. **同步执行**：目前不支持真正的后台任务（&）
3. **简单 cd 解析**：复杂的路径处理（如 `cd $(dirname $0)`）可能不准确
4. **无流式输出**：命令完成后一次性返回结果
