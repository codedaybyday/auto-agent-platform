## 🎓 学习资源

### 如何使用 Playwright 的 waitUntil

```typescript
// 可用的 waitUntil 选项:
'load'               // 等待 load 事件 (推荐)
'domcontentloaded'   // 等待 DOM 解析完成 (推荐)
'networkidle'        // 等待所有网络停止 (不推荐)
'networkidle2'       // 等待 ≤ 2 个网络连接
```

### 为什么选择 domcontentloaded

- ✅ 快速: 通常 1-2 秒
- ✅ 安全: 页面完全可交互
- ✅ 稳定: 网络友好
- ✅ 通用: 适用于大多数网站

### 为什么避免 networkidle

- ❌ 慢: 可能 10-30+ 秒
- ❌ 脆弱: 后台请求导致超时
- ❌ 不必要: DOM 已加载，不需要等待所有资源
