# 环境部署

将当前分支发布到测试环境，调用 `/mep-talos` skill 执行完整部署流程。

---

## 前置依赖

使用此命令需要安装 Talos CLI：

```bash
npm install -g @ee/talos-cli --registry=http://r.npm.sankuai.com
```

检查是否已安装：

```bash
talos --version
```

如果未安装或命令不存在，提示用户安装后再继续。

---

## 参数

- `$ARGUMENTS`：应用名或应用 ID（可选，不填则进入交互选择）

---

## 执行步骤

直接调用 `/mep-talos` skill 执行完整部署流程：

1. **检测 ONES 开发任务**：检查 `.catpaw/ones.json` 配置
2. **获取应用信息**：查询当前仓库关联的 Talos 应用
3. **获取发布模板**：列出可用模板供用户选择
4. **泳道名称确认**：泳道模板时询问泳道名称
5. **确认发布信息**：展示发布配置等待用户确认
6. **执行发布**：调用 Talos CLI 执行部署
7. **跟踪进度**：实时展示发布日志
8. **输出结果**：展示访问地址和后续建议

---

## 详细说明

完整的部署流程、CLI 命令参数、错误处理等详见 `/mep-talos` skill。
